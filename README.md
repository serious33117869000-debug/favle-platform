# Favle Platform - منصة Favle

منصة اجتماعية-تجارية مخصصة لمطوري الألعاب المستقلين. تمكن المطورين من تشكيل دوائر تعاون صغيرة (2-5 أعضاء) بهدف الترويج المتبادل وزيادة اكتشاف الألعاب.

## المميزات

### ✅ نظام المصادقة
- تسجيل الدخول بالبريد الإلكتروني
- تسجيل الدخول عبر OAuth (Steam / Discord) - Mock
- إنشاء حساب جديد

### ✅ البروفايل والمطورين
- صفحة بروفايل كاملة مع معلومات شخصية
- معرض للألعاب (صور/فيديو/روابط المتاجر)
- إضافة وتعديل الألعاب

### ✅ نظام الدوائر (Circles)
- إنشاء دائرة جديدة (اسم، وصف، قواعد)
- دعوة أعضاء عبر رابط
- حد أقصى 5 أعضاء لكل دائرة
- صفحة دائرة تعرض: الأعضاء، ألعابهم، الحملات النشطة

### ✅ Campaign Builder
- إنشاء حملة إطلاق لعبة داخل الدائرة
- تحديد تاريخ الإطلاق
- تعيين مهام لكل عضو (مشاركة، رتويت، خصم، Bundle)
- تتبع حالة المهام (Pending / Done)
- شاشة لعرض أداء الحملة باستخدام بيانات Mock

### ✅ صفحة الاكتشاف (Discovery)
- استعراض الألعاب والدوائر
- فلاتر حسب النوع، المنصة، الشعبية
- بحث في المحتوى

### ✅ الرسائل والإشعارات
- إشعارات داخلية
- نظام مراسلة مباشر بين أعضاء الدائرة

### ✅ لوحة الإدارة (Admin Panel)
- عرض الدوائر
- مراجعة المحتوى
- تتبع التقارير

### ✅ Onboarding
- دليل تفاعلي للمستخدمين الجدد
- خطوات واضحة للبدء

## التقنيات المستخدمة

- **Next.js 16** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Zustand** - State Management
- **Lucide React** - Icons
- **Mock API** - للوظائف الأساسية

## التثبيت والتشغيل

```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم التطويري
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل الإنتاج
npm start
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.
